Let's get you set up.

Your license is in your inbox and on your dashboard. Three quick steps and you'll be placing mockups in Photoshop.

Go to your dashboard
1

Grab your license + download

Sign in to your dashboard with the email you used at checkout. We'll send you a one-time link — no password.

On your dashboard you'll find:

  • Your license key — copy it for the activation step
  • The MockVault plugin download (latest version)
  • A list of devices you've activated MockVault on
Tip: We also emailed your license key with the receipt. Check the email titled "Your MockVault license key" if you'd rather find it there.
2

Install the plugin

Make sure Adobe Creative Cloud Desktop is open and running.

  1. Double-click the MockVault.ccx file you just downloaded.
  2. Creative Cloud will pop up an install prompt — click Install.
  3. Open or restart Adobe Photoshop.
  4. In the Photoshop menu bar, go to Plugins → MockVault to open the panel.
macOS only: If you see a Gatekeeper warning, right-click the .ccx file and choose Open instead of double-clicking.
3

Activate your license

The first time you open MockVault, you'll see the Activate MockVault screen.

  1. Paste your license key into the field.
  2. Click Activate.

That's it. MockVault verifies your key and loads every mockup in the library. You only do this once per device — install on up to 3 machines.

Need to free up a slot? Head to your dashboard and deactivate any device you no longer use.

Troubleshooting
MockVault doesn't appear in the Plugins menu

Make sure Creative Cloud shows MockVault as Installed. Fully quit and relaunch Photoshop. Confirm you're on Photoshop 2021 (22.0) or newer via Help → About Photoshop.

Creative Cloud didn't prompt to install

Make sure Creative Cloud Desktop is running first, then double-click the .ccx again. Alternative: Creative Cloud → gear icon → Manage Plugins → Install from Disk.

"Invalid license key" on activation

Copy the full key from your purchase email — no extra spaces. If you've lost the email, sign in to your dashboard to retrieve it.

Activation screen keeps reappearing

The plugin's local storage may have been cleared. Re-enter your license key — you can reuse it any time.

Mockups won't load / blank grid

Check your internet connection and click the refresh button (↻) in the panel. If it still misbehaves, restart Photoshop.

Placing a mockup does nothing

You need a document open in Photoshop before placing. Create or open a file first.

I only have Photoshop Beta installed — MockVault won't show up

The plugin is registered through the stable Photoshop release, so you need Photoshop 22.0+ (the regular, non-Beta build) installed alongside the Beta. Once both are installed, MockVault will be available in both. You don't have to use the stable version — it just needs to be present.
